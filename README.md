# Tienda - Prueba tecnica React

### Instrucciones para correr el proyecto en local.

## Docker

1 - Armar la imagen docker con docker compose:

```bash
docker compose build
```

2 - Correr la app con docker compose:

```bash
docker compose run
```

## Local

1 - Instalar dependencias con npm o pnpm:

```bash
npm install
```

2 - Correr aplicacion:

```bash
npm run dev
# o
pnpm dev
```

El proyecto estara disponible localmente en: http://localhost:5173/

> Los productos mockeados son usados de la API https://fakestoreapi.com/
